Book Finder App

Description

A web application that utilizes the Google Books API to recommend books to users based on user side input criteria. Working as a member of a four person team, I was responsible for coding front-end Javascript, including dynamic text and image population and click events, as well as polishing up CSS using the Bulma CSS framework
