Whiskey Journal

Description

A CRUD application that allows users to keep track of whiskeys theyâ€™ve tasted, assigning names, whiskey categories, tasting notes, and ratings via an express-handlebars based user interface that passes information into the back-end to be stored in a MySQL database with a PUT request. Users can then view their stored whiskey data with a GET request that dynamically populates in a responsive, mobile friendly format. As part of a three person team, I was responsible for designing a responsive front end, utilizing Express-Handlebars as a templating engine to pass user input between the front and back-end.
